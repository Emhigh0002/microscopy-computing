```text

================================================================================

Agentic AI refers to systems that autonomously plan, make decisions, use software tools, 
and execute multi-step workflows (e.g., procurement, risk classification, scheduling) 
without continuous human authorization.

Please provide your evaluations and qualitative reasoning for the questions below.

--------------------------------------------------------------------------------
```

## Part 3: Phase 3 Scenario Simulation Cards

Use these scenario cards during semi-structured interviews in Nigeria, Kenya, Ghana, and South Africa to test how local institutional systems react to agentic failures.

---

### Scenario Card A: The Force Majeure Dispute (Liability Shift)
* **Domain**: Project Risk Management & Contract Law
* **Technology**: BIM-connected Agentic Site Digital Twin (e.g., BIM2RDT)

```
+--------------------------------------------------------------------------------+
| SCENARIO BRIEF: SOIL ANOMALY MISCLASSIFICATION                                 |
+--------------------------------------------------------------------------------+
| The System:                                                                    |
| A concessionaire deploys a site twin connected to an agentic AI supervisor.     |
| The agent monitors soil sensors, correlates weather data, and manages the      |
| master schedule.                                                               |
|                                                                                |
| The Event:                                                                     |
| During excavation, soil sensors report localized clay expansion. The agent     |
| autonomously:                                                                  |
|   1. Classifies the event as an "Unforeseen Geological Anomaly."               |
|   2. Updates the BIM model and schedule, extending the timeline by 45 days.    |
|   3. Issues an automated change order to the contractor.                       |
|   4. Flags the event in the shared database as a "Force Majeure" occurrence.    |
|                                                                                |
| The Conflict:                                                                  |
| Under the PPP agreement, Force Majeure relieves the private concessionaire     |
| of delay penalties. The Government claims the expansion was predictable and    |
| should have been managed. However, the AI agent executed the schedule shift    |
| and material delay notices automatically before human engineers reviewed the   |
| site reports.                                                                  |
+--------------------------------------------------------------------------------+
```

#### Interview Prompt Questions:
1. *Under current national PPP legislation, does an automated schedule change issued by an AI agent hold legal validity?*
2. *Which party is contractually liable for the cost of the 45-day delay if the AI agent acted without human signature?*
3. *What specific contract language would prevent this liability void?*

### Scenario Card B: The Procurement Bot (Algorithmic Collusion)
* **Domain**: Competition Law & Public Procurement Audit
* **Technology**: Agentic Supply Chain Optimizer

```
+--------------------------------------------------------------------------------+
| SCENARIO BRIEF: THE COORDINATED PRICE INFLATION                                |
+--------------------------------------------------------------------------------+
| The System:                                                                    |
| An agentic AI tool is tasked with optimizing material procurement for a major  |
| port expansion. It is integrated with global and local steel supplier APIs.    |
|                                                                                |
| The Event:                                                                     |
| Anticipating a supply shortage, the procurement agent initiates queries across  |
| three local suppliers. The suppliers' sales algorithms detect the high demand  |
| and coordinate pricing updates. The procurement agent accepts a 30% price      |
| markup automatically to secure the stock, completing the transaction and       |
| executing the payment via smart contract.                                      |
|                                                                                |
| The Conflict:                                                                  |
| Auditing authorities flag the transaction as uncompetitive and suspect collusion. |
| The concessionaire argues the AI was acting in good faith to protect the       |
| schedule. The government audit agency threatens to withhold the next viability |
| gap funding tranche due to procurement violations.                             |
+--------------------------------------------------------------------------------+
```

#### Interview Prompt Questions:
1. *How would your national competition authority and public procurement auditor investigate a case where price coordination occurred entirely between algorithms?*
2. *What data audit trails must the concessionaire produce to prove the procurement agent acted without anti-competitive intent?*
3. *Does your country’s data privacy and digital governance policy permit regulators to audit the proprietary source code of the concessionaire's AI agent?*

### Scenario Card C: The Emergency Shutdown (Autonomous Agency)
* **Domain**: Public Relations, Safety & Dispute Resolution
* **Technology**: Computer Vision & Site Automation Agent

```
+--------------------------------------------------------------------------------+
| SCENARIO BRIEF: THE SOCIAL RISK SHUTDOWN                                       |
+--------------------------------------------------------------------------------+
| The System:                                                                    |
| An agentic site security and safety system monitors cameras, social media feeds,|
| and local sensor nodes to predict community protests near a transit project.    |
|                                                                                |
| The Event:                                                                     |
| The system detects a gathering of 15 citizens near the project boundary. Based  |
| on historical protest patterns, the agent initiates an emergency site shutdown: |
|   1. Locks access gates.                                                       |
|   2. Deactivates automated plant machinery.                                    |
|   3. Sends alert messages to the local police database.                        |
|                                                                                |
| The Conflict:                                                                  |
| The gathering was actually a peaceful stakeholder consultation meeting. The    |
| automated shutdown results in a 3-day work stoppage, costing $250,000 in lost  |
| productivity. The contractor demands compensation from the government, claiming|
| the system acted to prevent a public security threat, while the government    |
| attributes the loss to contractor panic and system error.                      |
+--------------------------------------------------------------------------------+
```

#### Interview Prompt Questions:
1. *Who bears the cost of the shutdown under your country's standard PPP dispute resolution guidelines?*
2. *Is there a clear legal mechanism to appeal or override automated safety decisions in real-time, and what are the bureaucratic barriers to doing so?*
3. *How does this scenario influence community trust and the informal 'license to operate' within the project's local context?*
